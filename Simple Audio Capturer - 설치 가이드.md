# Simple Audio Capturer - 설치 가이드

## 확장 프로그램 설치 방법

### 방법 1: Chrome 웹 스토어에서 설치 (배포 후)
1. Chrome 웹 스토어에서 "Simple Audio Capturer"를 검색합니다.
2. 확장 프로그램 페이지에서 "Chrome에 추가" 버튼을 클릭합니다.
3. 확인 대화 상자에서 "확장 프로그램 추가"를 클릭합니다.

### 방법 2: 개발자 모드로 직접 설치 (ZIP 파일 사용)
1. `simple-audio-capturer.zip` 파일을 다운로드하고 원하는 위치에 압축을 풉니다.
2. Chrome 브라우저를 열고 주소창에 `chrome://extensions/`를 입력합니다.
3. 오른쪽 상단의 "개발자 모드" 토글을 활성화합니다.
4. "압축해제된 확장 프로그램을 로드합니다" 버튼을 클릭합니다.
5. 압축을 푼 폴더를 선택하고 "폴더 선택" 버튼을 클릭합니다.

## 사용 방법

1. Chrome 브라우저 오른쪽 상단의 확장 프로그램 아이콘 중 "Simple Audio Capturer" 아이콘을 클릭합니다.
2. 팝업 창에서 "녹음 시작" 버튼을 클릭하여 현재 탭의 오디오 녹음을 시작합니다.
3. 녹음 중에는 상태 표시기에 녹음 중임이 표시됩니다.
4. 녹음을 중지하려면 "녹음 중지" 버튼을 클릭합니다.
5. 녹음이 중지되면 WAV 파일이 자동으로 다운로드됩니다.

## 주의 사항

- 이 확장 프로그램은 현재 활성화된 탭의 오디오만 녹음합니다.
- 녹음을 시작하기 전에 녹음하려는 탭이 활성화되어 있는지 확인하세요.
- 녹음된 파일은 WAV 형식으로 저장됩니다.
- 개인 정보 보호를 위해 다른 사람의 음성을 녹음할 때는 사전 동의를 구하세요.

## 문제 해결

- 녹음이 시작되지 않는 경우: 브라우저 권한 설정을 확인하세요.
- 오디오가 녹음되지 않는 경우: 해당 탭에서 오디오가 재생 중인지 확인하세요.
- 확장 프로그램이 작동하지 않는 경우: 브라우저를 재시작하거나 확장 프로그램을 재설치해보세요.

## 향후 업데이트 예정 기능

- 녹음 타이머 및 파형 시각화
- 다양한 오디오 포맷 지원 (MP3, WAV 등)
- 클라우드 저장 기능
- 실시간 음성 인식 API(STT) 연동
